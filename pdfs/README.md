# PDFs Folder

Place your PDF files in this folder to use them with the chatbot.

## How to Use

1. Copy your PDF files into this `pdfs` folder
2. Run the Streamlit app: `streamlit run app/main.py`
3. In the sidebar, select a PDF from the dropdown
4. Click "Load PDF" to load it as context
5. Start chatting - the LLM will use the PDF content to answer your questions

## Example

If you place a file called `research_paper.pdf` in this folder, it will appear in the PDF selection dropdown in the app.
